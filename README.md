# Task 2 - Phishing Email Analysis

## Objective

The objective of this task was to analyze a phishing email sample and identify common phishing indicators.

## Tools Used

- Sample phishing email
- Web browser
- GitHub

## Phishing Indicators Found

- Urgent payment notification
- Bank impersonation
- Suspicious verification request
- Generic recipient
- Hidden or suspicious links
- Unverified sender identity

## Conclusion

This task helped me understand how phishing emails use social engineering techniques to trick users into revealing sensitive information. By checking the sender, links, and content carefully, phishing attempts can be identified more effectively.
